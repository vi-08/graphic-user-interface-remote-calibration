# graphic-user-interface-remote-calibration
For syncing secondary laboratories to A*STAR NMC

Commands to run on terminal: 

[1] git clone https://github.com/vi-08/graphic-user-interface-remote-calibration.git

[2] cd graphic-user-interface-remote-calibration

[3] pip install flask

[4] python app.py

[5] Run local host link on browser
